# Guidelines

このディレクトリでは、`Auto Research Digest` を継続運用するための調査・編集ルールを管理します。

- `information-sourcing-policy.md`
  公開前の根拠確認、採用可能なソース、除外対象、公開ページ上の出典表示ルールを定義します。
- `editorial-expression-guideline.md`
  経営層向けニュースサイトとしての文体、トーン、中立性、図解の見せ方、避ける表現を定義します。

運用上の原則は次の 3 点です。

1. 話題の探索は広く行ってよいが、公開時の根拠は公式ドキュメントか論文に限定する。
2. 新規記事または大幅改稿に進む前に、対象トピックの公式ドキュメントまたは論文 URL を 20 件以上収集する。
3. 配信ページでは、判断材料と推測を混同させず、公開根拠欄にニュースサイトやブログなど第三者著作物を載せない。
